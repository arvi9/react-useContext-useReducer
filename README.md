# React: using useContext and useReducer
React Example: useContext and useReducer and webAPI

### Development

#### Using Docker compose for startup Postgresql
    ```
    docker-compose up
    ```


#### Client App
+ Start app
    ```
    $cd client-app
    npm start
    ```

+ Build App
    ```
    $cd client-app
    npm build
    ```

#### API
Using .Net5 Core Web API

+ Swagger API
    ```
    http://localhost:49764/swagger/index.html
    ```
